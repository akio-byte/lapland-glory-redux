# Lapland Glory 1995 – Redux

Uusi, moderni versio Lapin Glory 1995 -pelistä.

- Frontend: React + Vite + TypeScript
- Tavoite: selkeä DAY → NIGHT → MORNING -looppi, LAI-indeksi ja useita endingejä
- Vanha versio (referenssi): https://github.com/akio-byte/lapin-glory-1995

## Roadmap (lyhyesti)

1. Game engine v0.1 (`useGameEngine` + peruslooppi)
2. Event- ja ending-data erillisinä tiedostoina
3. Moderni UI (responsiivinen, Lapland AI Lab -teema)
4. (Valinnainen) Firebase/Cloudflare-telemetria

Tämä repo on koodipohja; narratiivinen sisältö ja eventit generoidaan AI-työkaluilla (Gemini 3 Pro, ChatGPT, Codex).
